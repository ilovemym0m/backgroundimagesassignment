The Tag Meaning
The "p" Tag! This tag is to write paragraphs and have them show on the website!
The "table" tag! This is the tag being used to write THIS table right now!
The "style" tag! This tag helps make the code less bland!
The "strong" tag! The text makes certain texts BOLD!
The "em" tag! This tag makes text italic!
The "ul, ol, and li" tag! This tag helps list words or sentences!
The h1-h6 tags! These tags are used for titles and subtitles(the one on top of the webpage right now)!
The "img" tags! This tag helps add images to a webpage or website!
The "nav" tags! This tag helps define a group of navigation links!
The "a href" tags! This tag helps add links to a webpage or website!
The "br" tags! This tag specifies a specific line break in the website/webpage text!
The "!DOCTYPE" tag! This tag states what coding language to use for the code!
The "html" tag! This tag helps convert HTML documents into webpages!
The "body" tag! This tag is where all the "p", "img", and "a href" tags go!
